# BBP quiz — obrazki na karty wyniku

Zdjęcia pojawiają się TYLKO na karcie wyniku (hero board), per styl (6 estetyk).
Pytania quizu nie używają realnych zdjęć — więc folder jest per STYL, nie per pytanie.

## Struktura

Jeden folder na styl, **4 zdjęcia** każdy. Nazwy slotów stałe (kod buduje URL-e
automatycznie, bez ręcznej edycji):

```
clinical/   quiet/   botanical/   editorial/   playful/   heritage/
  01-packaging.jpg
  02-product.jpg
  03-campaign.jpg
  04-detail.jpg
```

## Zasada doboru (variety)

4 RÓŻNE ujęcia na styl, nigdy warianty jednego produktu:
- 01 — opakowanie / packaging
- 02 — inny produkt z linii
- 03 — ujęcie kampanijne / model / scena
- 04 — detal / tekstura / makro

## Format

- JPG, kwadrat lub 3:4, ~1200px dłuższy bok, < 400 KB (karta jest mała).
- Spójne ze stylem (clinical = jasne/chłodne, heritage = ciemniejsze/cieplejsze itd.).
